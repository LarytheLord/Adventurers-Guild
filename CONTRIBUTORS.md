## How We Recognise Contributors

### NSoC 2026 (Nexus Spring of Code)
The programme runs **Apr 15 – May 30, 2026**. All issues labelled `NSoC-2026` are eligible for XP awards. Points are awarded for:

| Contribution | XP |
|-------------|-----|
| Bug report (accepted and fixed) | 5 XP |
| Good first issue / documentation | 10 XP |
| F-rank issue merged | 15 XP |
| E-rank issue merged | 25 XP |
| D-rank issue merged | 40 XP |
| C-rank issue merged | 60 XP |
| B-rank issue merged | 100 XP |
| A/S-rank issue merged | 150 XP |

XP is tallied in issue #184 (NSoC Contribution Tracking). Results are published at the end of the programme.

### Recognition
- Merged contributors are added to this file
- Bug reporters are named in issue comments and commit references
- Significant contributors are invited to continue as long-term collaborators
- Top contributors are mentioned in project announcements

### Path to Maintainer
Sustained, high-quality contributions across 3+ merged PRs in different areas of the codebase → invite to co-maintainer role. See [MAINTAINERS.md](MAINTAINERS.md) for what that entails.
